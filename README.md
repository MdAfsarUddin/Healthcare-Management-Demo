# 🏥 Healthcare Management System (Demo)

Enterprise-level Healthcare ERP solution developed using ASP.NET Core and Blazor, designed to support scalable hospital and healthcare operations.

---

# 🚀 Features

- Patient Registration
- Emergency Management
- Inpatient & Outpatient Modules
- Pharmacy Management
- Radiology & Laboratory
- Billing & Inventory
- Medical Records
- Role-Based Access Control
- Modular ERP Architecture

---

# 🛠 Technologies Used

- ASP.NET Core
- Blazor
- C#
- SQL Server
- REST API
- Entity Framework
- HTML / CSS / JavaScript

---

# 🏗 Architecture

This project follows a layered enterprise architecture:

```text
Client (Blazor UI)
        ↓
REST API Services
        ↓
Business Logic Layer
        ↓
Data Access Layer
        ↓
SQL Server Database
```
